TDD

Todo teste tem a padrão AAA
A -> Arrange = Onde vc prepara os dados para o testes
A -> Act = Onde executa a acção do teste
A -> Assert = Onde verificar se o teste executou corretamente


- Os metodos de teste sempre seram void e publicos.



CODE COVERAGE 
Verifica se seu código está sendo todo testado, aparece como vermelho o que está sem a cobertura para o teste, e verde o que está sendo testado pela classe de teste.
